# 지도 마커 모달 인터랙션 강화 계획

## 🎯 목표
- 지도 마커 클릭 시 발생하는 화장실 정보 모달(`ToiletPopup`)의 버튼 및 요소들에 최신 트렌드에 맞는 인터랙티브한 효과를 추가하여 사용자 경험을 고도화합니다.

## 🛠 작업 단계

### Phase 1: `ToiletPopup.tsx` 모달 내부 버튼 강화
- [ ] 일반 `button` 태그를 `motion.button`으로 교체합니다.
- [ ] 즐겨찾기(Favorite) 버튼에 호버 시 확장, 클릭 시 축소 및 색상 강조 효과를 추가합니다.
* [ ] 리뷰 관련 버튼(작성, 전체 보기)에 부드러운 배경색 전환 및 스케일 애니메이션을 적용합니다.
* [ ] 길찾기 버튼(카카오, 네이버)에 미세한 회전 및 그림자 효과를 추가하여 역동성을 부여합니다.
* [ ] 메인 방문 인증 버튼에 강렬한 호버 그림자 및 그라데이션 시각 효과를 강화합니다.

### Phase 2: 시각적 디테일 개선
- [ ] 모달 창의 `transition` 수치를 조정하여 더 쫀득하고 부드러운 반응을 구현합니다.
- [ ] 버튼 클릭 시 미세한 피드백을 주는 `whileTap` 속성을 표준화합니다.

### Phase 3: 최종 확인
- [ ] 프론트엔드 서버 상에서 실제 인터랙션이 매끄럽게 작동하는지 확인합니다.
- [ ] `docs/frontend-modification-history.md`에 UI 개선 사항을 기록합니다.
