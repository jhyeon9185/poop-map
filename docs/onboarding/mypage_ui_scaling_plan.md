# 마이페이지 UI 스케일 업 및 리팩토링 계획서

## 1. 개요
마이페이지의 주요 UI 요소와 폰트 크기를 전체적으로 확대하여 가독성과 시각적 임팩트를 개선하고, 불필요한 섹션을 제거하여 사용자 경험을 최적화함.

## 2. 주요 변경 사항

### 2.1 메인 컨테이너 및 공통 요소
- **컨테이너 너비 확대:** `max-w-3xl`에서 `max-w-4xl`로 변경하여 더 넓은 공간 확보.
- **탭 바(TabBar) 강화:** 아이콘 크기 확대(18->22), 패딩 및 간격 증가, 폰트 사이즈 확대(sm->base).

### 2.2 홈(Home) 탭
- **섹션 제거:** '오늘의 미션' 및 '오늘의 럭키 컬러' 섹션 삭제.
- **아바타 및 레벨 바 확대:** 아바타 크기 및 레벨 바의 두께와 텍스트 크기를 대폭 확대.
- **아이템 커스터마이징:** 중앙 Carousel 카드 크기 확대(180x240), 버튼 및 텍스트 스케일 업.
- **AI 분석 섹션:** 텍스트 크기 확대 및 카드 패딩 증가로 가독성 개선.

### 2.3 컬렉션(Collection) 탭
- **대시보드 리뉴얼:** 트로피 아이콘 및 달성률 지표 크기 확대.
- **칭호 도감:** Carousel 카드 크기 확대(180x240) 및 전체적인 그리드 간격 최적화.

### 2.4 리포트(Report) 탭
- **서브 탭 내비게이션:** 버튼 크기 및 폰트 확대.
- **데일리/주간 리포트:** 차트 및 분석 텍스트 크기 확대, PRO 안내 카드 시각적 강화.
- **잠금 오버레이:** 더 크고 명확한 모달 디자인 적용.

### 2.5 설정(Settings) 탭
- **목록 아이템 확대:** 아이콘 및 텍스트 폰트 크기 증가, 클릭 영역(패딩) 확대.
- **로그아웃 버튼:** 더 눈에 띄는 디자인으로 변경.

## 3. 기술적 세부 사항
- **framer-motion:** 탭 전환 및 요소 애니메이션의 물리적 거리(y, x)를 스케일에 맞춰 조정.
- **lucide-react:** 모든 주요 아이콘의 `size` 속성을 20% 이상 확대 적용.
- **Tailwind CSS:** `p-8` -> `p-12`, `text-xs` -> `text-sm`, `text-lg` -> `text-2xl` 등 전반적인 유틸리티 클래스 상향 조정.

## 4. 기대 효과
- 프리미엄한 디자인 느낌 강조.
- 모바일 및 데스크톱 환경 모두에서 향상된 가독성 제공.
- 불필요한 정보 제거로 핵심 기능(AI 분석, 아바타 꾸미기) 집중도 향상.
