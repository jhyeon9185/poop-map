# HeroSection 리디자인 구현 계획

## 1. 개요
현재의 HeroSection을 더 프리미엄하고 인터랙티브한 디자인으로 업데이트하기 위한 3가지 독립적인 디자인 옵션을 구현합니다.

## 2. 디자인 옵션 구성

### 안건 1: [Glass Depth] - 입체적 레이어 디자인
- **컨셉**: 다중 유리 레이어와 시차 효과(Parallax)를 활용한 공간감 강조.
- **주요 효과**: 마우스 움직임에 따라 배경의 빛무리(NovaGlow)와 UI 카드가 서로 다른 속도로 이동.
- **적합성**: 프리미엄하고 기술적인 느낌을 원하는 경우.

### 안건 2: [Organic Flow] - 유동체 시각화 디자인
- **컨셉**: 생동감 있는 유동체(Fluid) 배경과 과감한 타이포그래피의 조화.
- **주요 효과**: Fluid Canvas 배경, 한 글자씩 타자 치듯 나타나는 애니메이션, 스크롤 반응형 마스킹.
- **적합성**: 창의적이고 역동적인 브랜드 이미지를 전달하고 싶은 경우.

### 안건 3: [Health Dash] - 데이터 대시보드 디자인
- **컨셉**: 서비스가 제공하는 핵심 데이터를 미리 보여주는 기능 중심 레이아웃.
- **주요 효과**: 화면의 절반은 타이포그래피, 절반은 실시간 상태를 보여주는 위젯 모형 배치.
- **적합성**: 실제 기능성과 데이터 중심 앱임을 즉시 각인시키고 싶은 경우.

## 3. 구현 단계
1. `frontend/src/components/hero/` 디렉토리에 안건별 컴포넌트 파일 생성.
2. 각 옵션에 최적화된 애니메이션 프레임워크(Framer Motion) 적용.
3. `MainPage.tsx`에서 이들을 고를 수 있는 방식 가이드 제공.
